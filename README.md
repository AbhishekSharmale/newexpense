# Expense Tracker SaaS - Indian UPI Focus

AI-powered expense tracking for Indian UPI users with intelligent spending insights.

## Quick Start

### Frontend (React/Next.js)
```bash
cd frontend
npm install
npm run dev
```

### Backend (Node.js/Express)
```bash
cd backend
npm install
npm start
```
